# Sorpresa Vivis

Repositorio híbrido para migrar un prototipo HTML+JavaScript a una base escalable en Godot 4.

## Estructura principal
- `legacy_web/`: prototipo original preservado como referencia histórica y funcional.
- `godot_game/`: nueva base modular en Godot 4 + GDScript para continuar el desarrollo serio.
- `docs/`: documentación táctica de migración y decisiones.

## Dirección actual del proyecto
La dirección oficial de desarrollo es **Godot 4**, con enfoque **data-driven** y módulos separados para overworld, combate, narrativa, UI y persistencia.

## Documentación clave
- Migración accionable desde el prototipo: `docs/migracion_desde_legacy_web.md`.
- Guía de estructura técnica Godot: `godot_game/README.md`.
